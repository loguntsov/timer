Функции предоставляемые ТАЙМЕРОМ

Эти функции вы можете использовать в своей программе для организации взаимодействия ПЛАГИНА и ТАЙМЕРА.

void WINAPI StickFormResize(HWND Form,int NewWidth,int NewHeight);

Вызывается при изменении размеров формы плагина

Эта функция должна использоваться если вы используете прилипающие формы, и ваша форма изменила размеры.

Параметры:

Form - указатель на вашу форму

NewWidth,NewHeight - соответственно новые длина и высота формы

void WINAPI StickFormMove(HWND Form,int NewLeft,int NewTop);

Вызывается при перемещении формы плагина.

Функция должна вызываться перемещения вашей прилипающей формы.

Параметры:

Form - указатель на вашу форму

NewLeft,NewTop - соответственно новые координаты X и Y левого верхнего угла формы.

void WINAPI StickFormEndMove(void);

Вы должны использовать эту функцию, чтобы указать Таймеру, что вы не будете больше перемещать прилипающую форму.

void WINAPI AddStickForm(HWND Form);

Добавление формы, которая прилипает к основной форме таймера.

Параметры:

Form - указатель на вашу форму.

void WINAPI DeleteStickForm(HWND Form);

Удаление формы, прилипающей к основной форме таймера.

Параметры:

Form - указатель на вашу форму.

void WINAPI ViewFormAfterTime(char * FileNameToPlay,HDC HBitmap,char * Str);

Выводит стандартную форму оповещения Таймера.

Параметры:

FileNameToPlay - путь к файлу, который нужно проиграть

HBitmap - указатель на контекст устроства в котором хранится фон формы.

Str - строка которая будет мерцать в форме

char * WINAPI GetRegistryPath(void);

Возвращает местоположение настроек плагинов для таймера.в реестре.

HWND WINAPI GetHWNDMainTimerForm(void);

Возвращает HWND главной формы Таймера.

void WINAPI WebBrowserOpenURL(char * URL);

Открывает Web Browser Таймера на заданном URL.

char * WINAPI GetPluginDir(void);

Возвращает путь в каталог плагинов Таймера.

Желательно использовать эту функцию вместо GetCurrentDir(), которая возвращает неправильный путь.

void WINAPI ButtonStartClick(void);

Эмулирует нажатие кнопки ПУСК

Аналогично, если вы нажмете кнопку ПУСК Таймера

void WINAPI ButtonStopClick(void);

Эмулирует нажатие кнопки СТОП

Аналогично, если вы нажмете кнопку СТОП Таймера.
