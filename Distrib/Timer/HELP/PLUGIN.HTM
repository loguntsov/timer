Возможности плагинов для Таймера

Таймер предоставляет значительные средства для подключения плагинов, это могут быть манипуляции с формой таймера, его настройками,  с формой отображения «Время истекло !!!» и т.д.

Примеры  плагинов содержаться в дистрибутиве, их исходники можно скачать с сайта программы.

Плагины в дистрибутиве:

TimerGluk – скругление главного окна Таймера.

MultiTimer – позволяет контролировать несколько временных интервалов.
