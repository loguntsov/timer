Функции предоставляемые ПЛАГИНОМ

Вызовы этих функций Таймер использует для выполнения действия реализованных в плагине. Эти функции являются не обязятельными, т.е. какой-то функции может и не бить в вашем плагине.

int WINAPI _export VersionTimer(void)

Возвращает минимальную версия Таймера, в которой работает плагин.

bool WINAPI _export Activate(void)

Вызывается один раз, во время активации плагина. В этой функции вы должны написать код, который однократно инициализирует плагин.

Возвращает код продолжения:

true - продолжать работу плагина.

false - прекратить работу плагина.

По умолчанию должен использоваться true

bool WINAPI _export Close(void)

Вызывается один раз, при закрытии плагина. В этой функции вы должны описать код готовящий плагин к удалению из памяти.

Возвращает код продолжения:

true - продолжить удаление плагина

false - плагин удалять нельзя.

По умолчанию должен использоваться true

void WINAPI _export Configure(void)

Эта функция предназначена для конфигурирования (настройки) плагина пользователем, т.е. при вызове этой функции должна вызываться форма с определенными настройками плагина.

Если настройки не используются, то можно не включать вызов этой функции.

void WINAPI _export AboutPlugin(void)

Функция должна выдавать на экран информацию о плагине.

int WINAPI _export RunPlugin(int Tim,char *WaveFile)

Вызывается для передачи управления плагину по оповещению пользователя об истечении времени.

Параметры:

Tim - количество секунд, на которые был поставлен Таймер.

WaveFile - файл для проигрывания

char * WINAPI _export PluginName(void)

Возвращает текстовую строку с наименованием плагина.

void WINAPI _export TimerEvent(void)

Вызывается таймером каждую секунду во время своей работы.

bool WINAPI _export StartClick(void)

Вызывается Таймером при нажатии пользователем на кнопку ПУСК, или вызова функции ButtonStartClick

Возвращаемые значения:

true - продолжать процедуру нажатия кнопки ПУСК

false - закончить процедуру остановом.

bool WINAPI _export StopClick(void)

Вызывается Таймером при нажатии пользователем на кнопку СТОП, или вызова функции ButtonStopClick

Возвращаесые значения:

true - продолжать процедуру нажатия кнопки СТОП

false - закончить процедуру остановом.

void WINAPI _export DropFiles(char *Str)

Вызывается тамером, если на его галвную форму "кинули" (Drag-and-Drop) файл

Параметры:

Str - путь к файлу, который кинули.
