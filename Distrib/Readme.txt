Timer - программа для отмера заданного времени.
Возможности:
	1. Отмер заданного времени, после чего происходит выдача сообщения.
	2. Удобный графический интерфейс, скины.
	3. Поддержка подключаемых модулей (плагинов).
	4. Архивация скинов.
	5. Многоязыковый интерфейс

Возможности плагинов:
Плагины предназначены для расширения вохможностей основной программы Timer

Ниже представлены плагины входящие в стандартную поставку:
TimerGluk - скругление основной формы таймера.
MultiTmer - таймер может отмерять сразу несколько промежутков.
RunProgram - отключение запущенной программы, отключение компьютера 
	 спустя некоторое время.
